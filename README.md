# TENG-Based Gesture Recognition using Deep Learning

## Overview

This project presents a **Triboelectric Nanogenerator (TENG) based gesture recognition system** that converts mechanical energy into electrical signals and uses **deep learning (1D CNN)** to classify gestures.

The system records voltage signals generated from mechanical interactions and uses a trained CNN model to recognize different gestures.

---

## Features

- Converts **mechanical energy → electrical signal**
- Signal processing from **TENG voltage output**
- **1D Convolutional Neural Network (CNN)** for gesture classification
- **Test-Time Augmentation (TTA)** for improved prediction accuracy
- Confusion matrix and performance evaluation
- Achieves **~99% validation accuracy**

---

## Gesture Classes

The system recognizes the following gestures:

```
ASCEND
DESCEND
DIRECTION
HELP
LOW AIR
NO WORD
OK
OUT OF AIR
PROBLEM
STOP
```

---

## Model Architecture

The system uses a **1D Convolutional Neural Network**:

```
Input: Voltage Signal (16000 samples)

Conv1D (64 filters)
BatchNorm
ReLU
MaxPool

Conv1D (128 filters)
BatchNorm
ReLU
MaxPool

Conv1D (256 filters)
BatchNorm
ReLU
MaxPool

Conv1D (512 filters)
BatchNorm
ReLU
MaxPool

AdaptiveAvgPool

Fully Connected Layers
512 → 256 → Output Classes
```

## Dataset

The dataset consists of **CSV files containing voltage signals generated by a TENG device**.

Each file contains:

- Time column
- Voltage values

Example folder structure:

```
Train/
   ASCEND/
   DESCEND/
   HELP/
   OK/
   ...
```

Due to dataset size limitations, only a **sample dataset** is included in this repository.

## Training the Model

Run:

```bash

```

This will:

- Train the CNN model
- Save trained weights
- Generate accuracy plots
- Generate confusion matrix

---

## Testing the Model

Run:

```bash
python test_model.py
```

This will:

- Load the trained model
- Evaluate on test dataset
- Print per-class performance
- Generate confusion matrix
